Development of a game inspired by the course of CrieSeusJogos using the Unity engine and the C# language. At this point, is already implemented a basis scenario and characters, collisors, movement (idle, walk, run, roll, cut trees, dig and watering) and NPC walking in the city (AI). The dialog can already check the language of the game. Trees' drops are implemented and a leafes particle system for the hits at it. Player can now create a initial farm.
